# Time Tracker Azure Functions

## Requirements

- NodeJS 12

## Considerations

The functions run in **Serverless Framework** and use **Windows** like OS.

## Install

Install Serverless

```bash
npm install -g serverless
```

Check Install

```bash
serverless -v
```

Install azure plugin

```bash
serverless plugin install --name serverless-azure-functions
```
